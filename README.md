# Drawing App

## Project Overview

Drawing App is an interactive web application built using HTML, CSS, and JavaScript that allows users to create drawings directly on a digital canvas. The application provides a smooth drawing experience with customizable brush settings and canvas controls.

This project demonstrates the use of the HTML5 Canvas API, event handling, and interactive user interface development.

## Features

* Freehand drawing on canvas
* Custom brush color selection
* Adjustable brush size
* Clear canvas functionality
* Smooth drawing experience
* Interactive and user-friendly interface
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API

## How to Use

1. Open the application in your browser.
2. Select your preferred brush color.
3. Adjust the brush size if available.
4. Click and drag on the canvas to draw.
5. Use the clear button to reset the canvas.
